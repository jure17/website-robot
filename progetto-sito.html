cos'è un sito web o sito internet?

Un sito web o sito Internet(anche abbreviato in sito se chiaro il contesto informatico)
è un insieme di pagine web correlate, ovvero una struttura ipertestuale di documenti che risiede su un server web.
Queste pagine vengono create grazie al codice HTML (Hyper Text Markup Language), l'HTML è un linguaggio di programmazione
(come anche javascript e altri ancora).
